# long-way-down
submission for onekey! Uses only the enter key to play. 

## Usage

- **Itch**

    This project can be accessed on [itch.io](https://uselesslyuseful.itch.io/long-way-down). There is currently no mobile support. 

- **Cloning**

   This game uses the pygame module. Use the package manager [pip](https://pip.pypa.io/en/stable/) to install pygame.

   ```bash
   pip install pygame
   ```

## Code

- **Structure**

    All code for the game is in main.py.

- **AI Usage**

    No AI was used during the duration of this project. 

## Features

- **Elevator**

    The elevator doors are controlled with the enter key. When opening/opened, enter closes, and vice versa. 

- **Interactions**

    There are currently three interactions: a normal worker (let in), a runner (chased by zombie, let in runner and close doors before zombie), and a zombie alone (keep door closed the entire time). Each floor operates on a fifteen second timer. Three lives are given to the player, every time a wrong decision is made, a life is taken away. If all three lives are used up, the game resets. 
    
    More interactions will be added soon.
